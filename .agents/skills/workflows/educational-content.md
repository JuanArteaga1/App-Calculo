# Flujo de Contenido Educativo

1. Confirmar unidad y tema en `src/data/temas.js`.
2. Revisar consistencia con `teoria.md` y docs relacionadas.
3. Corregir notacion matematica y codificacion de caracteres.
4. Mantener explicaciones progresivas: intuicion, definicion, ejemplo, aplicacion.
5. Agregar ejemplos con funcion simple antes de ejercicios complejos.
6. Validar que el contenido sea coherente con el chatbot y la graficacion.

## Checklist

- El titulo y la descripcion son claros.
- La notacion no esta corrupta.
- Los ejemplos tienen resultado verificable.
- Hay interpretacion grafica o practica cuando aplica.
- El texto evita tecnicismos innecesarios.
