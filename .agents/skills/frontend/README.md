# Frontend

Esta carpeta concentra las instrucciones de frontend para LimitsHub.

## Archivos

- `instructions.md`: reglas de implementacion para React/Vite, componentes, accesibilidad y calidad.
- `design-system.md`: direccion visual, layout, componentes y experiencia esperada.
- `color-palette.md`: paleta oficial, uso recomendado y tokens actuales.
- `responsive-accessibility.md`: reglas responsive y accesibilidad.

## Fuente del contexto

Estas instrucciones se basan en:

- `vite-project/src/index.css`
- `vite-project/estilo_pagina.md`
- Componentes y paginas actuales en `vite-project/src/`
- Contexto general en `../context/project-context.md`
