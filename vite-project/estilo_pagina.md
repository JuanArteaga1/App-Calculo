1. Concepto general del diseño

El sitio está diseñado como una landing page educativa moderna, enfocada en transmitir:

Profesionalismo
Confianza académica
Tecnología
Accesibilidad para estudiantes universitarios

El enfoque visual recuerda plataformas como Coursera, Platzi o Khan Academy, pero con una identidad más simple y universitaria.

2. Distribución / Estructura (Layout)
A. Navbar (Barra superior)
Características:
Fondo blanco limpio
Logo a la izquierda (ícono matemático + nombre “LimitsHub”)
Menú horizontal a la derecha
Opción “Inicio” resaltada como botón activo
Diseño:
Uso de espaciado amplio (whitespace)
Navegación minimalista
Íconos pequeños para reforzar cada sección
Muy orientado a UX moderna
Objetivo:

Da sensación de:
Orden + navegación intuitiva + plataforma tecnológica
